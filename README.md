# watgunmai
 
 ### Project setup
 * clone project and its submodules   
 `❯ git clone --recurse-submodules https://github.com/tiemfah/watgunmai.git`  
 * binstall backend dependencies  
 `❯ cd watgunmai-backend && npm i`  
 * read and finish env.example instruction  
 `❯ cat env.example`  
 * start project  
 `❯ node index.js`
